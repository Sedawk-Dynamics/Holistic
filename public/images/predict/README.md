# Card artwork — provenance

These 16 images were sourced via the Openverse API, filtered to **CC0 / Public
Domain Mark** only. Both waive copyright, so no attribution is legally required
and commercial use is permitted. They are recorded here anyway so the origin of
each file is traceable, and so you know exactly what to replace.

All were resized to 800x600 (smart-cropped to the focal point) and re-encoded as
mozjpeg q82 — 18 MB of originals became ~1 MB total.

## Replacing them

Drop a file with the same name into this folder. Nothing else to change — the
card picks it up. Any topic without a file falls back to the gradient tile and
its icon, so you can swap them in one at a time.
